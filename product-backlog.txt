5. Prioridades máximas do produto – BANCO DE DADOS

5.1 ÁREA ADMINISTRADOR
• CADASTRO DO ADMINISTRADOR: login com senha para segurança da loja, com linha de acesso restrita aos administradores do produto.
• CAMPO FINANCEIRO: área de balanços contábeis para verificação de despesas e receitas.
• CAMPO DE FORNECEDORES: área para cadastro e acompanhamento de contratos.
• CONTROLE DE PRODUTOS: área para cadastro de produtos da loja, com estipulação de preços.

5.2 ÁREA CLIENTE
• CATALOGO DE PRODUTOS: área para o cliente escolher os produtos que deseja.
• CAMPO DE COMPRA: área onde o cliente pode efetuar sua compra e pagamento.
• CAMPO DE CONTATOS: área pra o cliente acessar os diferentes meios de comunicação com a loja.

6. Funcionalidades do produto

6.1 User Stories

Quem ?	Administrador	

O que ?	Acessar histórico de vendas - Para que ? Para ter controle sobre os lucros
O que ?	Acessar fornecedores e contratos - Para que ? Para auxiliar no controle de estoque
O que ?	Cadastrar produtos - Para que ? Para ter produtos disponíveis no site para compra
O que ?Lista de produtos cadastrados - Para que ? Para auxiliar no controle de estoque
O que ?	Acessar lista de clientes cadastrados  - Para que ? Para ter um controle dos usuários que acessam o site
O que ?	Cadastrar uma conta e acessar o login - Para que ? Para conseguir acessar campo do administrador, fazer alterações e acessar informações

Quem ?	Cliente	

O que ?	Ver detalhes de um produto - Para que ? Para saber exatamente se é isso que quer comprar
O que ?	Ver detalhes da compra efetuada - Para que ? Para ter detalhes sobre forma de pagamento e recebimento do produto
O que ?	Cadastrar uma conta e acessar o login - Para que ? Para conseguir comprar na loja
O que ?	Acessar os contatos da loja - Para que ? Para tirar dúvidas que podem surgir ou fazer reclamações
O que ?	Colocar compras em um carrinho - Para que ? Para organizar as compras
O que ?	Escolher uma forma de pagamento - Para que ? Para finalizar a compra
O que ?	Conseguir mudar, se preferir, endereço de entrega - Para que ? Para deixar o cadastro atualizado
